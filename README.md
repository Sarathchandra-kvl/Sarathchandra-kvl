## ◼ Featured Projects

### [Retrievix-IITI](https://github.com/sarathchandrakvl07/Retrievix-IITI)
*Self-correcting multi-agent retrieval system.*
- **Tech Stack:** `Python` • `LangGraph` • `LLaMA-70B` • `FAISS` • `PyTorch`
- **Engineering:** Architected an orchestration backend across specialized agents with dynamic routing. Implemented Adaptive RAG via parallel in-memory vector searches and a critique loop to mitigate hallucinations.

### [Nestoria](https://github.com/sarathchandrakvl07/Nestoria)
*High-concurrency hotel booking platform backend.*
- **Tech Stack:** `Node.js` • `PostgreSQL` • `Express` • `JWT`
- **Engineering:** Designed a normalized 3NF PostgreSQL schema leveraging targeted indexes and row-level locks to mitigate race conditions. Engineered custom PL/pgSQL triggers for real-time Bayesian sentiment scoring, enforcing strict ACID properties.

### [HashBandits](https://github.com/sarathchandrakvl07/HashBandits)
*Secure on-chain DAO governance infrastructure.*
- **Tech Stack:** `Solidity` • `Hardhat` • `OpenZeppelin`
- **Engineering:** Built an optimized governance system with snapshot-based voting and timelocks to prevent flash-loan manipulation. Reduced gas costs by ~48% through struct packing and caching, validated by a comprehensive test suite with 100% coverage.
